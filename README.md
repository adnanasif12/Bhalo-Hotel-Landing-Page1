# Bhalohotel Landing Page

This project contains a modern, responsive landing page for Bhalohotel using React, HTML, and CSS for the frontend, and Node.js for the backend server.

## Structure
- `frontend/`: React app for the landing page
- `backend/`: Node.js server

## Setup

### Frontend
1. Navigate to `frontend`
2. Install dependencies: `npm install`
3. Start the app: `npm start`

### Backend
1. Navigate to `backend`
2. Install dependencies: `npm install`
3. Start the server: `node server.js`

## Features
- Responsive design
- React components
- Node.js backend connection
